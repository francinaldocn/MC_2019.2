Censo da Educação Superior 2017 - UFC/UFCG (Computação)¶

Cursos de computação das Universidades Federais de Campina Grande, UFCG, e do Ceará campus Fortaleza, UFC.

Sobre os dados¶

Os dados utilizados nessa análise são um recorte dos dados originais do Censo da Educação Superior 2017 que podem ser encontrados no link: http://portal.inep.gov.br/web/guest/microdados.

Os recortes dos dados são referentes aos cursos de Bacharelado em Ciência da Computação das Universidades Federais de Campina Grande - UFCG, campus Campina Grande e do Ceará - UFC, campus Fortaleza.

Nos dados estão presentes as seguintes variáves:

CO_IES - Código único de identificação da Instituição de Ensino Superior (IES)
CO_CURSO - Código único de identificação do curso gerado pelo E-MEC
CO_ALUNO - Código de identificação gerado pelo Inep para o aluno da educação superior
CO_ALUNO_CURSO - Código de identificação gerado pelo Inep para o vínculo do aluno ao curso
TP_COR_RACA - Código da cor/raça do aluno
TP_SEXO - Informa o sexo do aluno
NU_IDADE - Idade que o aluno completa no ano de referência do Censo
TP_NACIONALIDADE - Código da nacionalidade do aluno
TP_DEFICIENCIA - Informa se o aluno é uma pessoa com deficiência, transtorno global do desenvolvimento ou altas habilidades/superdotação
IN_RESERVA_VAGAS - Informa se o aluno participa de programa de reserva de vagas"
IN_APOIO_SOCIAL - Informa se o aluno recebe algum tipo de apoio social na forma de moradia, transporte, alimentação, material didático e bolsas (trabalho/permanência)
IN_ATIVIDADE_EXTRACURRICULAR - Informa se o aluno participa de algum tipo de atividade extracurricular (estágio não obrigatório, extensão, monitoria e pesquisa)
TP_ESCOLA_CONCLUSAO_ENS_MEDIO - Tipo de escola que o aluno concluiu ensino médio
IN_MATRICULA - Informa se o aluno é matriculado no curso
IN_CONCLUINTE - Informa se o aluno é concluinte
IN_INGRESSO_TOTAL - Informa se o aluno é ingressante no curso, não importando a forma de ingresso utilizada
NU_ANO_INGRESSO - Ano de ingresso do aluno no curso
NO_IES - Nome da IES
SG_IES - Sigla da IES
NO_CURSO - Nome do curso
NO_MUNICIPIO - Nome do Município da IES
NO_UF - Nome da Unidade da Federação da IES

# Criando um novo dataframe com a contagem dos alunos por instituição/sexo
alsx = pd.DataFrame(df.groupby(['SG_IES', 'TP_SEXO'])['CO_ALUNO'].count()).reset_index()

# Subistituíndo os valores 1 e 2 por feminino e masculino
alsx['TP_SEXO'] = np.where(alsx['TP_SEXO'] == 1, 'feminino', 'masculino')

# Renomeando as colunas
alsx.columns = ['IES', 'SEXO', 'QTDE']

# Criando uma coluna com a proporção de alunos por IES/SEXO
alsx['PROP'] = 0.0
for i in range(0, alsx.shape[0]):
    if alsx['IES'][i] == 'UFC':
        alsx['PROP'][i] = round((alsx['QTDE'][i]/(alsx.groupby('IES')['QTDE'].sum().values[0])*100),2)
    else:
        alsx['PROP'][i] = round((alsx['QTDE'][i]/(alsx.groupby('IES')['QTDE'].sum().values[1])*100),2)

    else:
        alrc['PROP'][i] = round((alrc['QTDE'][i]/(alrc.groupby('IES')['QTDE'].sum().values[1])*100),2)
